# text_read() validates path

    Code
      text_read(1L)
    Condition
      Error:
      ! 'path' must be a non-NA and non-empty character of length 1.

---

    Code
      text_read("a-non-existent-file.txt")
    Condition
      Error:
      ! 'path' 'a-non-existent-file.txt' does not exist, is a directory, or is not readable.
